# demo_hzm_vision_yolo
RoboMaster 2027 视觉组考核项目，基于YOLO算法实现目标检测，包含模型推理、图像预处理等功能模块，用于机器人视觉识别任务开发。
项目文件说明
- 训练代码：`yolo_train.py`
- 模型导出代码：`yolo_convert.py`
- 训练日志与可视化结果：`runs/` 文件夹
- Git LFS配置：已配置`.gitattributes`文件
- 模型文件：`best.pt` 和 `best.onnx`，附在 [Release v1.0](https://github.com/Starry315177/demo_hzm_vision_yolo/releases/tag/v1.0) 中
